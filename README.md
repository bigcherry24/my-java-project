## Getting Started

To build and run the application, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd my-java-project
   ```

2. **Build the project:**
   ```
   ./gradlew build
   ```

3. **Run the application:**
   ```
   ./gradlew run
   ```

## Running Tests

To run the tests, use the following command:

```
./gradlew test
```
